# Ridiculous Coding - A IntelliJ IDEA Plugin

Make your coding experience in IntelliJ 1000x more ridiculous. This is a spiritual
 successor to [Ridiculous Coding for VS Code](
 https://github.com/jotson/ridiculous_coding).

Available for automated installation through the [JETBRAINS Marketplace](
 https://plugins.jetbrains.com/plugin/30712-ridiculous-coding?noRedirect=true).

Source code available on [GitHub](
 https://github.com/the-jolly-green-bryant/ridiculous-coding-intellij-plugin).
